# MR FOCUS React Website

This version uses the component structure requested:

- all main website sections are inside `src/components/`
- each component has its own folder with `.jsx` and `.css`
- pages are only for About, Categories, and Contact
- home page is built directly in `App.jsx` using components

## Run

```bash
npm install
npm run dev
```

On Windows PowerShell, use:

```powershell
npm.cmd install
npm.cmd run dev
```

Copy your original assets folder into `public/assets`.
